    This is the program guide for LoneStarCon 3. It should be suitable for use on most browsers and devices, and will be made accessible with additional functionality and information during the con itself.

    Program and participant data will be kept updated as final changes and corrections are made. Also, the contents of this "Info" view will be updated with more information.

    Program and participant data were last updated .

    This guide will work in your browser even when you don't have an internet connection. You can also install it as a home screen app:

    Safari on iPhone/iPad
    Tap on (share), then tap Add to Home Screen.
    Chrome on Android
    Tap on (menu), then tap Add to Home Screen.
    Firefox on Android
    Tap on (menu), then tap Bookmark and select Options, and Add to Home Screen.
    IE on Windows Phone
    Tap on (menu), then tap Pin to Start
